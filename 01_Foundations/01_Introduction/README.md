### 2. Roles in Data Science
**File**: `01_ds_roles.md`

Detailed comparison:
- **Data Analyst**: Descriptive analytics, reporting, SQL, Excel
- **Data Scientist**: Predictive analytics, ML, statistics
- **ML Engineer**: Production ML, deployment, scalability
- **Data Engineer**: Data pipelines, ETL, big data infrastructure

| Role | Focus | Key Skills | Tools |
|------|-------|------------|-------|
| Data Analyst | Reporting & insights | SQL, Excel, Visualization | Tableau, Power BI |
| Data Scientist | Predictive modeling | Statistics, ML, Python | Python, R, Jupyter |
| ML Engineer | Production ML | Software engineering, MLOps | Docker, Kubernetes, Cloud |
| Data Engineer | Data infrastructure | ETL, databases | Spark, Airflow, SQL |

Career path dan transition antar roles.

### 3. Data Science Workflow
**File**: `01_ds_workflow.md`

Complete pipeline:
1. **Business Understanding**
   - Define problem
   - Success metrics
   - Stakeholder alignment

2. **Data Collection**
   - Sources identification
   - Data acquisition
   - Legal/ethical considerations

3. **Data Preparation**
   - Cleaning
   - Transformation
   - Feature engineering

4. **Exploratory Data Analysis**
   - Univariate analysis
   - Bivariate analysis
   - Pattern discovery

5. **Modeling**
   - Algorithm selection
   - Training
   - Hyperparameter tuning

6. **Evaluation**
   - Metrics calculation
   - Model comparison
   - Validation

7. **Deployment**
   - API creation
   - Monitoring setup
   - Maintenance

CRISP-DM methodology explained.

### 4. Tools & Ecosystem
**File**: `01_tools_ecosystem.md`

**Programming Languages:**
- Python (most popular for DS)
- R (statistical analysis)
- SQL (data querying)
- Julia (high-performance computing)

**Libraries & Frameworks:**
- NumPy, Pandas (data manipulation)
- Matplotlib, Seaborn (visualization)
- Scikit-learn (ML)
- TensorFlow, PyTorch (DL)
- XGBoost, LightGBM (gradient boosting)

**Development Environment:**
- Jupyter Notebooks
- VS Code
- PyCharm
- Google Colab

**Version Control:**
- Git & GitHub
- DVC (data versioning)

**MLOps:**
- MLflow (experiment tracking)
- Docker (containerization)
- Kubernetes (orchestration)
- Airflow (workflow management)

**Cloud Platforms:**
- AWS (SageMaker, EC2)
- GCP (Vertex AI, Cloud Run)
- Azure (Azure ML)

### 5. Data Scientist Mindset
**File**: `01_ds_mindset.md`

**Critical Thinking:**
- Question assumptions
- Validate results
- Look for biases

**Scientific Method:**
- Hypothesis formation
- Experimentation
- Evidence-based conclusions

**Business Acumen:**
- Understand business context
- Translate technical to business language
- Focus on business impact, not just accuracy

**Continuous Learning:**
- Field is rapidly evolving
- Stay updated with research
- Learn from community

**Ethical Considerations:**
- Bias in data and models
- Privacy and security
- Transparency and explainability
- Social impact

### 6. Mini Project: First Data Science Project
**File**: `01_mini_project.ipynb`

Simple end-to-end project:
- **Dataset**: Iris atau Titanic (simple, famous dataset)

- **Steps**:

  2. Quick EDA
  3. Simple preprocessing
  4. Train model (e.g., Decision Tree)
  5. Evaluate
  6. Interpret results

**Purpose**: Show the big picture, motivate learning journey
